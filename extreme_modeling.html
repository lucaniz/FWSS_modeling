This tool models scenarios that have not been observed on mainnet yet. Thresholds are derived by reading the contract source code line by line — not from empirical data. Two separate risks are tracked: (1) daily gas pressure on the network (base fee impact), and (2) single transaction size vs tipset budget (stuck transaction risk). A transaction exceeding the tipset budget (~24.5 Bgas) would be stuck forever.
